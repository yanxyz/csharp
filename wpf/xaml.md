# WPF XAML

[XAML in WPF](https://docs.microsoft.com/en-us/dotnet/framework/wpf/advanced/xaml-in-wpf)

XAML 为 XML 格式，文件扩展名为 `.xaml`。

object elements

```cs
<StackPanel>
  <Button Content="Click Me"/>
</StackPanel>
```

object elements 有些属性不能以 attribute 指定，需要以 property element 指定。
